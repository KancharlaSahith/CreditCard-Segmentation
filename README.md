# CreditCard-Segmentation
AllLife Bank aims to expand its credit card business and improve customer satisfaction. Using customer spending patterns, financial attributes, and service interaction data, a clustering model will segment customers to enable targeted marketing and better support.
## Skills & Tools Covered
Exploratory Data Analysis
Data Scaling
Clustering
K-Means Clustering
Hierarchical Clustering
Silhouette Score
Cophenetic Correlation
Dimensionality Reduction
Cluster Profiling
